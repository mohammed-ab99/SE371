HTML TAG DESCRIPTION
!DOCTYPE html This document is HTML5 document, It must only appear once, at the top of the page
head This root element or beginning of HTML code that is divided into two parts (1) head and (2) body. The visible or display part is body
meta metainformation about the page. Meta means data about data, that means it will include all the information about and required by the page
title there are two usage of title tag (1) in head it is conisder as title of the page shown in chrome or IE tab, (2) in the body it is shown as tool tip text then you define it as an attribute to tag
br the tag used for \n or new line
/br is same as br
b bold the text it is used inside some tags, after the tag ends there is no new line
i italic the text it is used inside some tags, after the tag ends there is no new line
u underline the text it is used inside some tags, after the tag ends there is no new line
